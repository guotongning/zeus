# 我的VUE练习项目

## 构建项目

- 创建项目目录
```text
# 项目名称：zeus
mkdir zeus
cd zeus
npm init -y
```
- 创建完成后
    - 创建完成后会看到控制台如下信息，就算是初始化成功了。
